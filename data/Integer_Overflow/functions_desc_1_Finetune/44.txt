Type of contract: Simple Token Contract
Purpose: To create a simple ERC20-based token with a specified symbol, name, and decimals.
Features:
1. Designed for use in Crowdsale contracts to be issued to buyers.
2. Implements the full ERC20 standard with additional functionality for approval and allowance management.
3. Acts as an interface to be used in other more complex contracts.
4. May be used with a regular wallet to list on cryptocurrency exchanges.
5. Enables token burning.
