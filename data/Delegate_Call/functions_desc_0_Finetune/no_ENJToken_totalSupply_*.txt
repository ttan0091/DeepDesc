Type of the contract: ERC20
        Purpose of the code: To provide a standardized interface for fungible tokens on the Ethereum blockchain
        Key functionalities: The contract specifies the functions and events that an ERC20 token must implement, including functions for checking the balance of an account, transferring tokens, and allowing tokens to be moved on behalf of a third party.
        Explanation of the totalSupply function: The totalSupply function returns the total number of tokens in circulation, which is set during the token's initialization and can be adjusted through other functions like mint and burn.
