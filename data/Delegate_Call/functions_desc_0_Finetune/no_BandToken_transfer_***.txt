Type of contract: ERC20 Base Contract
        Purpose: To create a standard ERC20 token with minter capabilities
        Operation: The contract implements the ERC20 standard with functions for transfer, approve, transferFrom, totalSupply, balanceOf, and allowance. It also includes functions for minting and burning tokens, which can only be called by an address with minter role. Additionally, it provides a transferAndCall function for transferring tokens and executing a function in the receiving contract.
