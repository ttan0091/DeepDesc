Type of contract: MultiChain Delegated Vault
Author: PolyCEX (https://github.com/PolyCEX)
Primary function: Onchain token handler managing crosschain token transfers
Key features: 
1. Manages bridged tokens from different chains
2. Utilizes hooks and handlers to oversee token transfers between chains
3. Implements a whitelist mechanism for handlers
4. Ensures token standards compatibility across chains
5. Facilitates the swap of tokens across chains for users
6. Focuses on security and trustless operation through smart contract design
