Type of contract: HorseTokenCrowdsale
Purpose of contract: Enable the crowd sale of HORSE tokens using USDC and track the funding status to release funds to the beneficiary.
Working process: The `checkGoalReached` function updates the `areFundsReleasedToBeneficiary` flag based on the current time compared to the funding start and end times. If the current time is within the funding period, the flag is set to false, indicating funds are not yet released.
