Type of contract/function: NFT auction system
    Purpose: To manage the auction and sale process of NFTs
    Participants/Entities: NFT owners, bidders
    Core functions:
        1. `startAuction`: Initiates a new auction for an NFT
        2. `placeBid`: Allows a bidder to place a bid on an ongoing auction
        3. `endAuction`: Finalizes an auction and transfers the NFT to the winning bidder
        4. `claimBack`: Allows the NFT owner to claim back the NFT if no bids were placed
        5. `isOnSale`: Checks if an NFT is currently up for sale
